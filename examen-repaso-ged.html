Examen de Repaso GED

Un examen de práctica completo de Razonamiento Matemático, diseñado para simular la experiencia del día del examen real. Incluye dos partes — sin calculadora y con calculadora — y una clave de respuestas con explicaciones paso a paso.

46 preguntas 115 minutos Ir a la clave de respuestas ↓

Antes de comenzar

  • Necesitarás un lápiz y una calculadora para realizar el examen (la calculadora solo se permite en la Parte 2).
  • Hay dos tipos de preguntas: de opción múltiple (elige la mejor respuesta entre 4) y de respuesta abierta (escribe tu respuesta directamente, sin opciones).
  • Está bien adivinar — no se pierden puntos por responder incorrectamente.
  • El examen real incluye una hoja de fórmulas geométricas y algebraicas, para que te enfoques en aplicar las fórmulas, no en memorizarlas.
  • Al terminar, revisa la clave de respuestas para ver en qué áreas necesitas reforzar.

Clave de respuestas y explicaciones

Revisa tus resultados para ver dónde te equivocaste y qué áreas necesitas reforzar.

1. Opción C — 472

11×36 + 6×12 + 4 = 472 pulgadas.

2. Opción D — 44

2x − 5 = 83 → 2x = 88 → x = 44.

3. Opción D

Cada descuento del 15% se aplica multiplicando por 0.85. El primer descuento: (200)(0.85) = 170. El segundo descuento del 15% se aplica de nuevo sobre ese resultado: (200)(0.85)(0.85).

4. Opción B — (−1, 3)

Sustituyendo cada punto en 2x+4y=10: solo (−1, 3) da 2(−1)+4(3) = 10.

5. Respuesta: −30

Sigue el orden de operaciones (PEMDAS): primero dentro del corchete, 4 + (22×5) = 4 + 110 = 114. Luego 114 ÷ 6 = 19. Finalmente: 5 + 8×(−2) − 19 = 5 − 16 − 19 = −30.

6. Opción B — 20%

De 2014 a 2015: (20000−16000)/20000 = 20%. De 2015 a 2016: (16000−12800)/16000 = 20%.

7. Opción D — 729 cm³

Ancho = 27÷3 = 9. Altura = 9÷3 = 3. Volumen = 27×9×3 = 729 cm³.

8. Opción D — 300%

0.60A = 0.20B → B = (0.60/0.20)A = 3A, es decir, el 300% de A.

9. Respuesta: 90

6 × 3 × 5 = 90 combinaciones posibles.

10. Opción C — 50 pies

En un triángulo 30°-60°-90°, el lado opuesto al ángulo de 30° es la mitad de la hipotenusa. Con el lado de 30 pies como el cateto opuesto a 30°, la hipotenusa (la escalera) mide 50 pies.

11. Opción C — 6

(24−x)÷x = 3 → 24−x = 3x → 24 = 4x → x = 6.

12. Opción B — 30

Ángulos suplementarios suman 180°. Si x + 5x = 180, entonces 6x = 180, x = 30.

13. Opción C — 5:9

Velocidad de John: 150÷6 = 25 km/h. Velocidad de Alice: 180÷4 = 45 km/h. La razón 25:45 se simplifica a 5:9.

14. Opción A — 10%

40% × 25% = 0.40 × 0.25 = 0.10 = 10%.

15. Respuesta: y = 5

Usando eliminación: multiplica −x+2y=10 por 3 → −3x+6y=30. Suma a 3x−4y=−20 → 2y=10 → y=5.

16. Opción C — 475 km

Primeras 5 horas: 40+45+50+35+55 = 225 km. Siguientes 5 horas: 50×5 = 250 km. Total: 225+250 = 475 km.

17. Opción C — 8 horas y 24 minutos

420÷50 = 8.4 horas. La parte decimal: 0.4×60 = 24 minutos.

18. Opción D — 8999

El número más pequeño de 4 dígitos es 1000; el más grande es 9999. La diferencia es 8999.

19. Respuesta: 1296

6⁴ = 6×6×6×6 = 1296.

20. Opción D — 10 cm

Por el Teorema de Pitágoras: 6²+8² = c² → 36+64 = 100 → c = 10.

21. Respuesta: 240

La razón 2:3 significa 2 de cada 5 estudiantes son chicos. 600÷5 = 120, y 120×2 = 240 chicos.

22. Opción C — 125%

25 = (porcentaje/100)×20 → porcentaje = 2500/20 = 125%.

23. Respuesta: 130

El lado faltante (altura) = 54 − 18 − 12 − 14 = 10. Área del trapecio = ½ × h × (b1+b2) = ½ × 10 × (12+14) = 130.

24. Opción C — 30

(2/3)×18 = (2/5)×x → 12 = (2/5)x → x = 30.

25. Opción B — 0.88 D

Disminución del 20%: D×0.80 = 0.80D. Aumento del 10% sobre ese valor: 0.80D×1.10 = 0.88D.

26. Opción B — 16π

Área = πr² → 64π = πr² → r² = 64 → r = 8. Circunferencia = 2πr = 2π(8) = 16π.

27. Opción B — 30%

(28−40)/40 × 100% = −30%, es decir, un descuento del 30%.

28. Opción A

El ingreso después de x años es 2000x + 24000. Un ingreso mayor que el promedio se expresa como I > 2000x + 24000.

29. Opción C — 140%

1.75 es ¿qué porcentaje de 1.25? 1.75 = (porcentaje/100)×1.25 → porcentaje = 140%.

30. Opción B — 50 millas

Por el Teorema de Pitágoras: 40²+30² = c² → 1600+900 = 2500 → c = 50 millas.

31. Opción D — 15

Números primos consecutivos: 3 y 5. Su producto: 3×5 = 15.

32. Opción B — 15%

530.40 es ¿qué porcentaje de 624? 530.40/624 = 0.85 = 85%. El descuento es 100%−85% = 15%.

33. Opción B — 15

Si Mia = 60, entonces Ava = 30 (la mitad de Mia). Emma es la mitad de Ava: 15.

34. Opción D — 17/18

Después de extraer 17 de 18 bolas, queda 1. La probabilidad de no haber sacado la bola marrón en ningún momento es 17/18.

35. Opción B — 36

Si x es el número más pequeño: 38 = [x+(x+1)+(x+2)+(x+3)+(x+4)]/5 → 190 = 5x+10 → x = 36.

36. Opción C — 18

Área del piso: 6×24 = 144 cm². Baldosas necesarias: 144÷8 = 18.

37. Opción C — 7.32 kg

12.2×600 = 7320 gramos. 7320÷1000 = 7.32 kg.

38. Opción C — 600 ml

4% de x = 24 ml → 0.04x = 24 → x = 600 ml.

39. Opción B — 61.28

Peso total de chicas: 18×60 = 1080 kg. Peso total de chicos: 32×62 = 1984 kg. Total: 3064 kg ÷ 50 estudiantes = 61.28 kg.

40. Opción C — 400

90% de x = 360 → x = 360÷0.90 = 400.

41. Opción B — 9

Ordenando: 4, 5, 8, 9, 13, 15, 18. El número del medio es 9.

42. Respuesta: 1004.8 pulgadas²

Área de superficie de un cilindro = 2πr(r+h) = 2(3.14)(8)(8+12) = 2(3.14)(8)(20) = 1004.8.

43. Respuesta: 24

18 = (13+15+20+x)/4 → 72 = 48+x → x = 24.

44. Respuesta: $560 (precio original)

75% de x = 420 → x = 420÷0.75 = 560.

45. Cómo verificar tu gráfica

Para cada región candidata, elige un punto y compruébalo en ambas desigualdades. Por ejemplo, el punto (−4, −4): −4 ≤ −4+4 → −4 ≤ 0 ✓, y 2(−4)+(−4) ≤ −4 → −12 ≤ −4 ✓. Ese punto satisface ambas desigualdades, así que su región es la solución correcta.

46. Respuesta: $1,800 de interés

Usa la fórmula de interés simple I = prt: I = (8000)(0.045)(5) = 1800.
